## QR Code Generator in Python
A simple Python program to generate QR codes for URLs, text, or any data you want. Perfect for quickly sharing information.

## Features
- Generate QR codes from URLs, text, or any input string.
- Save QR codes as image files (PNG).
- Easy-to-use command-line interface.
- Customizable QR code size and border.

## Requirements 
- Python 3.X
- qrcode library <br>
 Install the library using the following in terminal:<br>
 pip install qrcode

## Example Usage
Enter text or URL to generate QR code: https://github.com <br>
Enter the file name to save the QR code: github_qr <br>
QR code saved as 'github_qr.png'
